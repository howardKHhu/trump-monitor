隱私權政策

最後更新：2025 年 4 月

本網站（https://trump-monitor.pages.dev/，以下簡稱「本站」）致力於保護使用者隱私。本政策說明本站如何蒐集、使用及保護您的資料。

一、本站的性質

本站為純前端靜態網站，不設任何後端伺服器，不蒐集、儲存或處理任何個人識別資料。所有使用者操作（如 TACO 標記、翻譯快取）均儲存於您自己的瀏覽器本機儲存空間（localStorage），本站無法存取這些資料。

二、瀏覽器本機儲存（localStorage）

本站使用 localStorage 儲存以下資料，這些資料僅存在於您的裝置上：

  • trump_trans_v1:*：翻譯快取，避免重複呼叫翻譯 API
  • trump_taco_v1：您手動或系統自動標記的 TACO 候選與確認記錄

您可以隨時透過瀏覽器設定清除這些資料。

三、第三方服務

本站使用以下第三方服務，各自擁有獨立的隱私權政策：

  • Google Analytics 4（GA4）
    用於統計網站流量。Google 可能透過 Cookie 蒐集匿名瀏覽資料（頁面瀏覽次數、裝置類型、地區等）。詳見 Google 隱私權政策。
  • Google AdSense
    本站顯示 Google AdSense 廣告。Google 可能根據您先前的瀏覽行為提供個人化廣告，並使用 Cookie 達成此目的。您可至 Google 廣告設定 管理個人化廣告偏好。詳見 Google 廣告政策。
  • Factbase / Roll Call API
    本站透過此公開 API 取得川普在 Truth Social 的公開發文，不傳送任何使用者資料。
  • MyMemory 翻譯 API / Google Translate
    本站傳送貼文原文（英文）至翻譯 API 進行翻譯，不傳送任何個人識別資訊。

四、Cookie

本站本身不設置 Cookie。Google Analytics 及 Google AdSense 可能透過 Cookie 蒐集匿名統計資料及提供廣告服務。您可透過瀏覽器設定拒絕或刪除 Cookie。

五、兒童隱私

本站不針對 13 歲以下兒童提供服務，亦不蒐集兒童的個人資料。

六、政策更新

本站保留修改本隱私權政策的權利。重大變更將更新本頁面的「最後更新」日期。
